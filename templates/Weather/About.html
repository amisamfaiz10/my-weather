Welcome to the Weather App.


This app's backend is developed using Python(django) and the simple frontend is created using raw HTML and CSS. The app provides weather information for any city you specify, using the city's coordinates to fetch the weather information.


NOTE: Pictures are provided only for "Clear sky, Rain, Thunderstorm, Snow, Drizzle, Cloudy" weather descriptions.
